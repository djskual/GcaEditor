# Release Notes
## Added
- Added a Settings window with persistent user preferences for updates, editor behavior, viewer interaction, and window placement.

## Improved
- Improved update checking with optional automatic startup checks and prerelease filtering.
- Improved usability by allowing window size and position to be remembered between sessions.
- Improved viewer interaction by making horizontal trackpad scrolling configurable.
- Improved the Settings window with a more compact two-column layout using category navigation on the left and options on the right.
- Improved settings usability by reducing vertical clutter and making future options easier to scale.
- Improved editor safety by making delete confirmations configurable for zones and ambient images.
- Improved background loading behavior with a configurable auto-fit viewer option.
- 
## Fixed
- Fixed several Settings options that were displayed but not yet connected to runtime behavior.

## Cleanup
- Centralized app preference storage into a single persistent settings file.
